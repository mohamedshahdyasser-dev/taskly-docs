Taskly

Taskly is a simple, intuitive web-based task management app designed to help teams and individuals organize, track, and prioritize their work efficiently.


---

Features

Create Tasks: Add new tasks with due dates and priorities.

Organize Projects: Group tasks under different projects.

Task Status Tracking: Mark tasks as "To Do," "In Progress," or "Done."

Search & Filter: Quickly find tasks by project, priority, or status.

Responsive Design: Works smoothly on desktop and mobile devices.



---

Installation

Prerequisites

Node.js >= 18.x

npm >= 9.x


Steps

1. Clone the repository:



git clone https://github.com/your-username/taskly.git

2. Install dependencies:



cd taskly
npm install

3. Start the app locally:



npm start

4. Open your browser at http://localhost:3000




---

Usage

1. Create a new project: Click "Add Project," give it a name.


2. Add tasks: Inside a project, click "Add Task," set title, description, priority, and due date.


3. Track progress: Update task status as you work through your list.


4. Search & filter: Use the search bar or filter options to focus on specific tasks.
